## 前端运行

```bash
# 克隆项目
git clone http://192.168.18.166:3000/IT/Caile-ERP-Vue3

# 进入项目目录
cd Caile-ERP-Vue3

# 安装依赖
yarn --registry=https://registry.npmmirror.com

# 启动服务
yarn dev

# 构建测试环境 yarn build:stage
# 构建生产环境 yarn build:prod
# 前端访问地址 http://localhost:80


数据：/api/import/listPage/ImportMoney
字段：/api/import/getImportField/ImportMoney

```
